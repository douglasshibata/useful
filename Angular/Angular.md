## Comandos Angular

### Instalando NPM:
```
    sudo apt install npm
```

### Instalando Angular:
```
    npm install -g @angular/cli
```

### Inicia um projeto:
```bash
	ng new <NAME>
```

### Start angular server:
```bash
	ng serve --open(opcional para abrir automaticamente no browser)
```

### Gerar componente:
```bash
	ng generate component <NAME>
```

### Gerar service:
```bash
	ng generate service <NAME>
```
